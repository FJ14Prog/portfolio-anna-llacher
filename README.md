# Portfolio Anna Llacher Alsina

Portfolio professional d'Anna Llacher Alsina, doctoranda en Comunicació, productora audiovisual, docent universitària i professional de la comunicació.

## Descripció

Aquest projecte és una web portfolio personal desenvolupada amb HTML, CSS i JavaScript mínim.  
L’objectiu és presentar de manera clara, professional i visual la trajectòria d’Anna Llacher Alsina en l’àmbit de la comunicació, la producció audiovisual, la docència universitària i la recerca.
